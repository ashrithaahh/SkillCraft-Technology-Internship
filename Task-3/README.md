# Sudoku Solver

## SkillCraft Technology Internship - Task 03

### Description
A Java-based Sudoku Solver that automatically solves a 9x9 Sudoku puzzle using an optimized Backtracking Algorithm with constraint tracking.

### Features
- Automatic Sudoku solving
- Constraint-based optimization
- Input validation
- Conflict detection
- Execution time measurement
- Professional board formatting

### Technologies Used
- Java
- Arrays
- Recursion
- Backtracking Algorithm
