# 📂 Internship Tasks

## 🔹 Task 1 - Temperature Converter Pro

### Description
A Java application that converts temperatures between Celsius, Fahrenheit, and Kelvin scales.

### Features
- Celsius to Fahrenheit Conversion
- Fahrenheit to Celsius Conversion
- Celsius to Kelvin Conversion
- Kelvin to Celsius Conversion
- User-Friendly Interface
- Input Validation

### Skills Demonstrated
- Java Fundamentals
- Conditional Statements
- User Input Handling
- Mathematical Computations

---

## 🔹 Task 2 - Number Guessing Game

### Description
An interactive Java game where the system generates a random number and challenges the user to guess it.

### Features
- Random Number Generation
- User Guess Validation
- Hint System (Too High / Too Low)
- Attempt Tracking
- Interactive Console Experience

### Skills Demonstrated
- Java Random Class
- Loops
- Conditional Logic
- User Interaction Design

## 🔹 Task 3 - Sudoku Solver

### Description
A fully automated Sudoku solving system developed using an optimized Backtracking Algorithm.

The program accepts an incomplete Sudoku puzzle and efficiently computes the correct solution.

### Features
- Automatic Sudoku Solving
- Constraint-Based Optimization
- Input Validation
- Conflict Detection
- Execution Time Analysis
- Professional Board Formatting

### Skills Demonstrated
- Recursion
- Backtracking Algorithms
- Data Structures
- Performance Optimization
- Algorithm Design

---

## 🔹 Task 4 - Product Data Extraction & Analytics System

### Description
A Java-based data extraction and analytics application designed to collect product information from an e-commerce website and export structured datasets into CSV format.

The system processes product information and generates organized datasets for analysis.

### Features
- Product Data Extraction
- Product Name Collection
- Price Analysis
- Rating Extraction
- Review Count Processing
- CSV Export Functionality
- Data Analytics Reporting
- Execution Time Monitoring

### Skills Demonstrated
- Web Scraping
- Data Processing
- Object-Oriented Programming
- File Handling
- CSV Data Management
- Software Architecture Design
